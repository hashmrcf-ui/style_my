06 · Layout

Less is the RULE

Minimal text
title ≤6 words · one support line max. if it reads like a sentence, cut it.
CENTERED by default
heading + visual centered. size the stage columns to content - never strand a label against the edge.
No kickers by default
no module codes or section labels above the heading. one header per slide unless asked.
STATIC by default
print energy. the only motion: sparing dot-matrix animation.
No corner text
no watermarks, tags, slide numbers, or folio labels. clean canvas.
One idea per slide
one heading, one visual. split before you cram.
Mix the backgrounds
mostly paper · black for emphasis beats · half/half splits for variety.
Pixel font sparingly
Doto marks 1-2 emphasis words in a heading - never body text or whole lines.
